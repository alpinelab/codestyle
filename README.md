# AlpineLab code style

Code style used by AlpineLab in all our projects.

## Usage

### Rubocop integration

* Use this gem as a dependency, or just install it;
* Update Rubocop configuration with:
  ``` yaml
  inherit_gem:
    alpinelab-codestyle:
      - config/default.yml
  ```
